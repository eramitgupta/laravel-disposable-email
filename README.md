# Laravel Disposable Email Docs

This repository contains the documentation site for **Laravel Disposable Email**.

## About

Laravel Disposable Email is a Laravel package that helps detect and block disposable or temporary email addresses during validation and runtime checks.

It includes:

- built-in disposable domain detection
- validation rule support
- runtime checking
- Blade directive support
- remote syncing
- custom blacklist support
- optional caching
